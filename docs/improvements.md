## **향후 개선 계획**

- **테스트 및 코드 품질**
  - 주요 기능 테스트 커버리지 70% 이상 목표
  - 전역 예외 처리 추가 및 테스트 코드 보강
  - 설정 정보 보안 강화

- **인프라 및 성능**
  - Nginx 로드밸런싱 적용 및 무중단 배포
    - 로드밸런싱을 위한 백엔드 컨테이너 다중 운영 및 알고리즘 테스트 (round-robin, least_conn, ip_hash)
  - 경기 일정 관리 자동화
    - 일/주/월 단위 자동 갱신 스케줄링

- **인증 및 보안**
    - OAuth2 기반 소셜 로그인 구현
    - Redis 기반 인증 정보 캐싱
      - 멀티 디바이스 로그인 관리 (현재는 브라우저 쿠키 삭제만 지원)
      - 전체 기기 로그아웃, 비밀번호 변경 시 자동 전체 로그아웃
 
- **데이터 처리 확장**
    - 실시간 채팅 메시지 영구 저장
    - 관리자 기능 구현 (API 데이터 지연 시 수동 관리)
    - Riot API 추가 연동 (사용자 인게임 데이터)
