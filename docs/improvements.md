## **개선 방향**

- **경기 일정 관리 자동화**
    - 일/주/월 단위로 경기 일정 자동 갱신
    - 기존 일정과 비교해 변경 감지 후 자동 업데이트
 
- **관리자 기능 구현**
    - 리그/토너먼트/팀/경기일정 수동 관리 기능 구현 (API 데이터 지연 대응)

- **Nginx 로드밸런싱 적용 및 무중단 배포**
    - Nginx 로드밸런싱 알고리즘 테스트 (round-robin, least_conn, ip_hash)
    - 백엔드 Docker 컨테이너 2개 운영 테스트

- **실시간 채팅 데이터 저장**
    - 채팅 메시지 DB 저장

- **경기별 커뮤니티 기능 추가**
    - 경기별 채팅방 개설
    - 경기별 다이어리 기능 (자유 게시글)

- **소셜 로그인**
    - OAuth2 기반 소셜 로그인

- **Redis 기반 인증 정보 캐싱**
    - 소셜 로그인 인증 코드, 사용자 정보 캐시 처리
    - 멀티 디바이스 로그인 관리 및 로그아웃 처리 개선
        - 현재는 브라우저 쿠키 삭제 방식만 사용
        - 모든 기기에서 로그아웃 기능 별도 구현
        - 비밀번호 변경 시 자동 전체 로그아웃 기능 포함

- **Riot API 추가 연동**
  - 공식 경기 외 일반 사용자 인게임 데이터 수집
  - 전적 검색 또는 커뮤니티 기능 연계 목적
